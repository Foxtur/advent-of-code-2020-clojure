# AoC 2020 Clojure

Advent of Code 2020 solutions

## Usage

Run the project directly:

    $ clojure -M -m aoc2020.main

Run the project's tests (they'll fail until you edit them):

    $ clojure -M:test:runner

Build an uberjar:

    $ clojure -M:uberjar

Run that uberjar:

    $ java -jar aoc2020.jar
